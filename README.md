# Web-портфолио | Клиентская разработка

Добро пожаловать в моё веб-портфолио!  
Здесь собраны лабораторные работы и проекты по курсу **Разработка клиентской части веб-приложений**.  
Сайт построен на генераторе статических сайтов **Hugo**.

**Сайт доступен по ссылке:**  
[https://resurssss.github.io/web_porfolio/](https://resurssss.github.io/web_porfolio/)

---

## Технологии

- [Hugo](https://gohugo.io/) — статический генератор
- GitHub Pages — хостинг
- GitHub Actions — автоматический деплой
